# CamberFoil

![CamberFoil Preview](https://via.placeholder.com/800x400.png?text=CamberFoil+Aerodynamics+and+CFD)

CamberFoil is a high-performance web application designed for the rapid generation, visualization, and export of 2D and 3D airfoil geometries. It bridges the gap between aerodynamic theory and computational fluid dynamics (CFD) by allowing users to export directly to OpenFOAM and ParaView formats through a modern, Glassmorphism-styled interface.
